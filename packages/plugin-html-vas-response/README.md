# html-vas-response

## Overview

This plugin collects responses to an arbitrary HTML string using a point-and-click visual analogue scale.

## Loading

```js
<script src="https://unpkg.com/@jspsych-contrib/plugin-libet-intentional-binding@1.0.0">

```

## Compatibility

jsPsych v7.0
